### Convert Cyrillic to Utf-8
Eg convert:
```bash
\u0420\u0435\u0431\u044f\u0442\u0430
```
To:
```bash
Ребята
```
### Working
Put scitp to parent folder and run:
```bash
python convert.py
```
After script works, you will can view new folder:
```bash
converted-json
```
Converted files will has new extension - *.convert
